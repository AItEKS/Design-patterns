## Цезарь с курицей

### Ингредиенты:

| Ингредиент            | Количество       |
|-----------------------|------------------|
| Куриное филе          | 200 г            |
| Салат Романо          | 50 г          |
| Сыр Пармезан          | 50 г             |
| Чеснок                | 10 г       |
| Белый хлеб            | 30 г       |
| Соль                  | 5 г         |
| Черный перец          | 2 г         |
| Оливковое масло       | 10 мл           |
| Лимонный сок          | 5 мл          |
| Горчица дижонская     | 5 г          |
| Яйца                 | 2 шт.            |

### Пошаговый список:

1. **Подготовка ингредиентов:**
   - Нарежь куриное филе кубиками, наруби чеснок, нарежь хлеб на кубики.
   - Вымой салат, обсуши его.
   - Натри сыр Пармезан на терке.

2. **Жарка курицы:**
   - Обжарь на сковороде куриное филе с чесноком до готовности. Посоли и поперчи по вкусу.

3. **Жарка хлеба:**
   - На той же сковороде обжарь кубики хлеба до золотистости.

4. **Приготовление заправки:**
   - В миске смешай оливковое масло, лимонный сок, горчицу, измельченный чеснок, соль и перец.

5. **Сборка салата:**
   - В большой миске смешай кубики курицы, хлеба, листья салата. 
   - Полей заправкой и тщательно перемешай.

6. **Подача:**
   - Подай Цезарь на тарелку, посыпь сверху тертым сыром.
   - Укрась кружочками яиц.

7. **Приятного аппетита!**

### Время приготовления: 30 минут
